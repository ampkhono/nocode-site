## 🎮 3. ノーコードゲームエディタの実装

### 作成したファイル
- `editor.html` - メインのエディタページ
- `editor.js` - エディタ機能の実装
- `EDITOR_README.md` - エディタの使い方ガイド

### 実装機能

#### 1. 質問ノードの作成・編集
- 質問文の入力
- 選択肢の追加・削除・編集
- 各選択肢から次のノードへの分岐設定

#### 2. 結果ノードの作成
- 結果テキストの入力
- 画像ファイル名の設定
- URLとボタンテキストの設定

#### 3. データ管理
- JSON形式でのプロジェクト保存
- プロジェクトの読み込み機能
- ノード一覧の表示

#### 4. エクスポート機能
- CSV形式でのエクスポート（908.py互換形式）
- 質問と選択肢をCSV形式に変換

#### 5. プレビュー機能
- 選択中のノードのプレビュー表示
- 右サイドバーでリアルタイム確認

### UI構成
- **左サイドバー**: ノード一覧、操作ボタン
- **中央エリア**: エディタ（質問・結果の編集）
- **右サイドバー**: プレビュー表示

### データ構造

#### 質問ノード
```javascript
{
    id: "q_0",
    type: "question",
    title: "質問 1",
    text: "質問文",
    choices: [
        {
            text: "選択肢1",
            value: 0,
            nextId: "q_1" // 次のノードID
        }
    ]
}
```

#### 結果ノード
```javascript
{
    id: "r_0",
    type: "result",
    title: "結果 1",
    text: "診断結果",
    image: "attention_type.png",
    url: "https://example.com",
    buttonText: "公式サイトを見る"
}
```

### CSV出力形式
```
Start,"開始メッセージ"
Selection,"質問1","選択肢1","選択肢2"
Selection,"質問2","選択肢A","選択肢B"
Result,0,"結果1","image.png"
Result_URL,0,"結果2","ボタンテキスト","https://example.com"
End
```
